# Estudo de Caso – Análise Exploratória de Dados: Descobrindo Sinais de Compra em uma Plataforma de Cursos On-line

No mercado brasileiro de infoprodutos, é comum acumular milhares de respostas de formulários e dados comportamentais. O problema é que a maioria das equipes não sabe exatamente por quê, como e quando esses dados serão usados. Nem toda pergunta ou comportamento tem poder preditivo. Com método, é possível decidir o que vai ser mantido, descartado e testado.
Esse foi exatamente o objetivo deste estudo de caso realizado em uma plataforma de cursos online: identificar, antes de qualquer modelo preditivo, quais variáveis realmente importam para prever a compra.
